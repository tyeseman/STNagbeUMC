# STNagbeUMC
This is the official app of the Stephen Torwen Nagbe United Methodist Church
The app provide important informations for the order of service.
Begining with the Get Started page
The Home page consist of seven buttons with the preview of the church edifice.
The content of the app carries: the comminion bulletin, the apostle creed, prayer of illumination, benidiction, welcome & birthday song, offortory, etc.
